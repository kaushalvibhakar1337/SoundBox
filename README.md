
# SoundBox

A Web Music Player
## Tech Stack

**Client:** React, Redux, TailwindCSS and more.

**Server:** Firebase


## Run Locally

Clone the project

```bash
  git clone https://github.com/kaushalvibhakar1337/SoundBox
```

Go to the project directory

```bash
  cd SoundBox
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
